# TCC - Markowitz + Machine Learning

## Objetivo
Combinar previsão de retornos (ML) com otimização de portfólio regularizada.

## Métodos
- Lasso, Ridge, Elastic Net
- Markowitz com regularização L1 + L2
- Covariância via Ledoit-Wolf

## Métricas
- Sharpe Ratio
- Drawdown
